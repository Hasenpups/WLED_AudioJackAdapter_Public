# WLED_AudioJackAdapter

## Features
- Connect 3.5mm audio jack to WLED controller input pin

## BOM
- J1: Audio Jack PJ320D - https://www.ebay.de/itm/393422313034
- J2: 3-pin header male
- C1, C2: 100nF 0805
- R1, R2: 680 Ohm 0805
- R3, R4: 1M Ohm 0805
